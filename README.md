# pomodoropastatimer
This is a pomodoro study/ focus timer with a pasta sauce theme.

## Features
- 25-minute (unskippable) timer
- Start, pause, and reset buttons
- Visual countdown

## Built with
- HTML
- CSS
- JavaScript

## What I learned
- How to work with timers and basic math in JavaScript
- How to scale a project so that it doesn't become over-engineered and overwhelming
- How to update the DOM over time
